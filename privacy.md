# Privacy Policy for Free Games Advanced


## 1. Introduction
This Privacy Policy explains how the  Free Games Advanced Bot ("the Bot") handles information.

## 2. Information We Collect
The Bot may collect and store limited, non-personal information to function properly:
- Discord server (guild) IDs
- Channel IDs for giveaway notifications
- Basic metadata about posted giveaways (titles, links, timestamps)
- User IDs of administrators who configure the bot (for internal logging only)

No message content, personal data, or private Discord information is collected or stored.

## 3. How We Use the Data
The stored data is used **only** to:
- Remember which channel to post notifications in.
- Prevent duplicate posts.
- Allow administrators to resend or manage giveaways.

## 4. Data Storage
Data is stored securely in local files or a database managed by the bot’s developer.  
No data is shared, sold, or transferred to third parties.

## 5. Data Deletion
Server owners may contact the developer to:
- Remove their server’s stored configuration.
- Request deletion of all related data.

To request deletion, please contact:
- Email: [your@email.com]
- Discord: [Your Discord Tag]

## 6. External Services
The Bot uses the public GamerPower API (`https://gamerpower.com/api`) to retrieve free game information.  
We are not responsible for the content or policies of GamerPower or other external services.

## 7. Children’s Privacy
The Bot is not directed at children under 13.  
By using the Bot, you confirm compliance with Discord’s minimum age requirements.

## 8. Changes to This Policy
We may update this Privacy Policy from time to time. Continued use of the Bot after changes constitutes acceptance of the updated Policy.
